# Safeguard CHI

Two-stage ML guard pipeline for the GourmetGram food classification app, deployed as a Docker Compose stack on Chameleon Cloud.

Follow along at [Safeguard CHI](https://teaching-on-testbeds.github.io/safeguard-chi/).

This material uses: one `m1.medium` VM at KVM@TACC, and one floating IP.
